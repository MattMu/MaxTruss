## 代码编译说明
在代码目录下运行`make`命令，当前目录得到`kmax_truss`可执行程序。

## 运行使用说明
运行命令：
```shell script
./kmax_truss -f ***.tsv(图数据文件路径)
```
标准输出为：
```shell script
kmax = x, Edges in kmax-truss = y.
```
其中x、y为整数，x代表kmax-truss值，y代表kmax-truss中的边数量。
